# Student Management System (C++)

A console-based application to manage student records using Object-Oriented Programming and file handling.

## Features

- Add Student
- Search Student
- Update Student
- Delete Student
- Persistent storage using binary files
- Student count
- Top performer detection
- Sorting students by CGPA

## Technologies Used

- C++
- Object-Oriented Programming
- File Handling
- STL

## How to Run

Compile:

g++ src/main.cpp src/Student.cpp src/Database.cpp -o sms

Run:

./sms
